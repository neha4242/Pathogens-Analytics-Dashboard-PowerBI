# 🦠 Pathogens Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze pathogen-related data.

The dashboard helps visualize trends, KPIs, disease distribution, and geographical insights.

---

## Features

- Interactive Dashboard
- KPI Cards
- DAX Measures
- Power Query Transformations
- Trend Analysis
- Geographic Analysis
- Dynamic Filters
- Drill Through
- Business Intelligence Reporting

---

## Tools Used

- Microsoft Power BI
- DAX
- Power Query
- Excel
